Les techniques en escalade

  • Le crochetage

    Cette technique consiste à utiliser le bord d'un point d'appui pour se relever ou se stabiliser. Les crochets sont généralement utilisés sur les bords rocheux ou les points de crochet.

  • Le mantel

    Cette technique est utilisée pour surmonter la prise suspendue. Il s'agit de placer les mains sur les points de chargement, puis de soulever le corps avec les bras jusqu'à ce que les jambes puissent reposer sur les points de chargement.

  • Le dyno

    Cette technique est utilisée pour atteindre des points d'arrêt dans des positions statiques trop éloignées pour être atteintes. Cela implique de sauter et d'attraper une prise en l'air, puis de vous relever rapidement.

  • Le stem

    Cette technique est utilisée pour escalader les fissures en écartant les jambes et en les pressant contre les côtés de la fissure. En répartissant le poids du corps de part et d'autre du mur, la tige peut maintenir une position stable et avancer le long de la fissure.
